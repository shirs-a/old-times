## 安装jdk

1、使用如下命令查看可安装的jdk版本。      

```
yum -y list java*
```
<img src="/old-times/png/linux/1.png">


2、选择自己要安装的jdk版本即可。

```
yum install -y java-1.8.0-openjdk-devel.x86_64
```

3、查看jdk版本，安装成功

```
java -version
```

## 卸载jdk

1、查看已安装的jdk信息

```
yum list installed|grep java
```

<img src="/old-times/png/linux/2.png">

2、删除

```
yum -y remove java-latest-openjdk*
```
3、通过java -version查看卸载情况
